= GKStartingGuide =

Diese Seite bietet einen kurzen Überblick über die Funktionen des Plugins G.E.E.K.Edit. [[BR]]

G.E.E.K.Edit bietet die Möglichkeit in einem lokalen Repository Wikiseiten zu verwalten. Darunter fällt das Erstellen, [[BR]]
das Bearbeiten und das Löschen. Erstellt werden die Seiten im Textdateiformat. Der Wikisyntax ist dabei vom Benutzer einzuhalten. [[BR]]
[[BR]]
Das eigentliche Verwalten erfolgt durch die Zusammenarbeit von Git Befehlen und Trac API Funktionen. [[BR]]
[[BR]]
Um eine Wikiseite lokal zu erstellen wird eine Textdatei im Wikiornder innerhalb des Repositorys erstellt. Diese kann mit beliebigem Inhalt [[BR]]
unter Einhaltung des Wikisyntax gefüllt werden. Die Wahl des Editors obliegt dabei dem Benutzer. [[BR]]
Die Wikiseite wird anschließend mit einem Git push Befehl in das externe Repository geladen. Dabei erstellt G.E.E.K.Edit automatisch eine Wikiseite [[BR]
mit dem Namen und dem Inhalt der Textdatei. Beim Namen wird die Endung entfernt. Ein Bearbeiten und Löschen ist auf dieselbe Weise möglich. [[BR]]
[[BR]]
Wikiseiten die online über die Weboberfläche in Trac erstellt werden, werden ebenfalls von G.E.E.K.Edit automatisch in das mit Trac verbundene Repository [[BR]]
gespeichert. Von dort aus können sie mit Git pull in jede geklonte Arbeitskopie geladen und dort bearbeitet werden.[[BR]]
G.E.E.K.Edit aktualisiert die entsprechenden Textdateien der Wikiseiten bei folgenden Aktionen: [[BR]]
* Erstellen einer Wikiseite
* Bearbeiten einer Wikiseite
* Umbenennen einer Wikiseite
* Löschen der letzten Version einer Wikiseite
* Löschen einer Wikiseite
